# React Expense Tracker

A modern expense tracker built with React that allows users to manage budgets, add/remove expenses, and dynamically track balance updates.

Features

* Add a total budget
* Add expenses with title and amount
* Remove expenses
* Automatic balance updates
* Refund balance on expense removal
* Negative balance warning (turns red)
* Interactive and responsive UI

Tech Stack

* React
* JavaScript
* CSS
* Vite
